# Hotel Reservas - Sistema Full Stack

## Sobre o projeto
O **Hotel Reservas** é um sistema full stack desenvolvido como atividade acadêmica, com o objetivo de simular o gerenciamento de um hotel.

O sistema permite o cadastro de quartos e reservas, além da visualização e controle das informações de forma organizada através de uma interface web.

## Funcionalidades

### Gestão de Quartos
- Cadastro de novos quartos
- Listagem de quartos cadastrados
- Visualização de reservas por quarto
- Exclusão de quartos

### Gestão de Reservas
- Cadastro de reservas por quarto
- Informações do hóspede
- Data de entrada e saída
- Listagem de reservas do quarto
- Exclusão de reservas

## Fluxo do sistema
O fluxo de navegação segue a lógica:

**Lista de Quartos → Cadastro de Quarto → Reservas do Quarto → Cadastro de Reserva → Lista de Reservas**
